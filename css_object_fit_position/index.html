General note

I have made use of the :target pseudo class to create a CSS-only tabbed navigation. All the <article> "tabs" are absolutely positioned on top of one another and given opacity: 0; to start with, then a transition is used to make the tabs smoothly appear when the links that link to them are clicked on them, ie when they become a target (article[id*=tab]:target {}).
